## 3. Các thay đổi cần thực hiện

### 3.1 Thêm helper method `RenameBlocksInDb`
**Mục đích:** Đổi tên tất cả user-defined blocks trong sourceDb bằng prefix duy nhất (vd: `File2_`) để tránh trùng tên khi clone.

**Code mới:**
```csharp
private void RenameBlocksInDb(Database db, Transaction trans, string prefix)
{
    var bt = (BlockTable)trans.GetObject(db.BlockTableId, OpenMode.ForRead);
    foreach (ObjectId btrId in bt)
    {
        var btr = (BlockTableRecord)trans.GetObject(btrId, OpenMode.ForRead);
        // Chỉ rename user-defined blocks (không phải layout, không phải anonymous)
        if (btr.IsLayout || btr.IsAnonymous || btr.IsFromExternalReference)
            continue;

        string oldName = btr.Name;
        string newName = prefix + oldName;

        // Đổi tên (cần upgrade lên Write mode)
        var btrWrite = (BlockTableRecord)trans.GetObject(btrId, OpenMode.ForWrite);
        btrWrite.Name = newName;
        AcadLogger.LogInfo($"Renamed block: {oldName} → {newName}");
    }
}
```

**Nơi gọi:** Trong mỗi vòng lặp xử lý source file, sau khi mở file và bắt đầu transaction.

### 3.2 Cập nhật GetSourceLayout nhận layoutName
**Mục đích:** Hỗ trợ chỉ định layout cụ thể thay vì lấy đại layout đầu tiên.

**Thay đổi signature:**
```csharp
private Layout GetSourceLayout(Database db, Transaction trans, string desiredLayoutName)
```

**Logic mới:**
```csharp
if (!string.IsNullOrEmpty(desiredLayoutName))
{
    var layouts = (DBDictionary)trans.GetObject(db.LayoutDictionaryId, OpenMode.ForRead);
    if (layouts.Contains(desiredLayoutName))
    {
        var layoutId = layouts.GetAt(desiredLayoutName);
        return (Layout)trans.GetObject(layoutId, OpenMode.ForRead);
    }
    // Fallback: không tìm thấy → lấy layout đầu tiên
}
// Fallback: lấy layout đầu tiên không phải Model
```

**Nơi gọi cập nhật:** Cập nhật tất cả nơi gọi hàm này để truyền `source.Layout`.

### 3.3 Sửa MergeToSingleLayout offset tracking
**Mục đích:** Lưu lại idMap sau clone để có thể transform đúng vị trí.

**Thay đổi tại lines ~346-360:**
```csharp
// Sau khi clone xong, lưu ngay các destination IDs

foreach (ObjectId id in ids)
{
    if (idMap.Contains(id) && !idMap[id].Value.IsNull)
        clonedIds.Add(idMap[id].Value);
}
allClonedIds.Add(clonedIds);
```

**Cần thêm biến:** `List<List<ObjectId>> allClonedIds = new List<List<ObjectId>>();`

**Transform tại lines ~387-396:**
